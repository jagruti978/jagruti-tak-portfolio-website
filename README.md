# Jagruti Tak — Developer Portfolio

A modern and fully responsive portfolio website built using React.js, Tailwind CSS, and Framer Motion to showcase my projects, skills, education, internships, and achievements.

## What I Learned
- Building reusable React components
- Responsive web design with Tailwind CSS
- Smooth animations using Framer Motion
- Client-side routing with React Router DOM
- Form handling and EmailJS integration
- UI/UX design and glassmorphism effects
- Mobile responsiveness and layout optimization
- Clean project structure and component architecture

## Technologies Used
- React.js
- Tailwind CSS
- Framer Motion
- React Router DOM
- EmailJS
- React Icons

## Features
- Fully Responsive Design
- Animated UI Sections
- Dynamic Project Pages
- Achivements and certificates
- Interactive Education Timeline
- Internship Showcase
- Contact Form with Toast Notifications
- Modern Glassmorphism UI
